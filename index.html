⚙️ TERMINAL MANUAL & PEDOMAN

1. TAB: PENARIKAN ADMIN

  1. Salin seluruh log teks transaksi penarikan dari grup/sistem Anda.
  2. Tempel teks tersebut ke dalam **Kotak Area Teks** (kolom kosong besar teratas). Proses pemisahan data akan berjalan otomatis.
  3. Klik tombol **"SALIN DATA UTAMA"** untuk mengambil hasil bersih yang terstruktur, atau **"HAPUS SEMUA DATA"** untuk mengosongkan riwayat.

2. TAB: ADMIN QRIS PGA

  1. Siapkan file laporan berformat `.csv`.
  2. Tarik & taruh file tersebut ke kotak putus-putus, atau klik area **"TARIK & LEPAS FILE CSV / KLIK UNTUK MEMILIH"** untuk memilih berkas dari folder perangkat Anda.
  3. Gunakan tombol **"SALIN ID & NOMINAL"** atau **"SALIN NOMOR REFERENSI"** untuk menyalin matriks data yang diinginkan secara instan.

3. TAB: PANEL QRIS CEPAT

  1. Jika ingin memfilter data lama, salin ID Transaksi terakhir Anda lalu tempel di kolom **"PELACAK FILTER TRANSAKSI TERAKHIR"**.
  2. Salin seluruh isi log teks dari panel admin Anda, kemudian tempel di kotak **"LOG MASUKAN PANEL"** bawahnya.
  3. Klik **"SALIN DATA UTAMA PANEL"** atau **"SALIN SEMUA ID TX"** untuk mengekstrak data log yang valid.

4. TAB: PGA MOTION

  1. Salin baris teks transaksi dari log PGA Motion yang berisi struktur kata kunci "NOTE".
  2. Tempel langsung ke dalam kotak teks **"MASUKAN PGA MOTION"**.
  3. Klik **"SALIN DATA UTAMA MOTION"** untuk memindahkan tumpukan data bersih ke papan klip Anda.
